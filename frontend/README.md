# Industrial L&D AI Assistant - Frontend

An AI-powered Learning & Development (L&D) Assistant designed to help organizations improve workforce training, skill development, and knowledge management through Generative AI.

## 🚀 Features

- Secure Authentication
- Personalized Learning Dashboard
- AI-Powered Learning Assistant
- Skill Gap Analysis
- Learning Analytics & Insights
- Knowledge Repository
- Training Recommendations
- Modern Responsive UI
- Role-Based Access Control

---

## 🛠️ Tech Stack

### Frontend
- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Recharts
- Lucide React

### Backend
- FastAPI
- Python
- Machine Learning Models
- REST APIs

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/RishuRaj2507/smartlearn-ai.git
```

### Navigate to Frontend

```bash
cd smartlearn-ai/frontend
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Application will be available at:

```text
http://localhost:3000
```

---

## 🔧 Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_API_URL=http://localhost:8000
```
